# 🚀 Projeto em Destaque

## Predição de Sinistros em Seguros de Automóveis

### 🎯 Objetivo
Desenvolver um pipeline de dados robusto para prever a ocorrência de sinistros em apólices de seguro, utilizando técnicas estatísticas e de Machine Learning com arquitetura Bronze, Silver e Gold.

---

## 🛠 Tecnologias Utilizadas

- Python  
- SQL  
- Databricks  
- GLM  
- Random Forest  
- Gradient Boosted Trees (GBT)  

---

## 🔍 Principais Etapas do Projeto

### 🔹 Camada Bronze – Preparação
- Limpeza e padronização de dados  
- Tratamento de valores ausentes  
- Análise e controle de outliers  
- Normalização de variáveis categóricas  

### 🔹 Camada Silver – Engenharia e Modelagem Inicial
- EDA completa  
- Seleção de features  
- Tratamento de desbalanceamento  
- Testes com GLM, Random Forest e GBT  
- Avaliação com AUC, Recall, Precisão e F1-score  

### 🔹 Camada Gold – Otimização
- Ajuste de `classWeightCol` para tratar desbalanceamento  
- Ajuste de threshold (0.4) para aumento de sensibilidade  
- Avaliação com matriz de confusão  
- Comparação final entre modelos  

---

## 📊 Resultados

O modelo **Gradient Boosted Trees (GBT)** apresentou o melhor equilíbrio entre precisão e recall, demonstrando maior capacidade de capturar padrões complexos e identificar sinistros mesmo em cenário de forte desbalanceamento.
